<task type="auto">
  <name>Task 2: Create diagonal section CSS classes</name>
  <files>C:/Users/bskim/Downloads/beamacademy-index.html</files>
  <action>
Add CSS classes for diagonal sections. Place these after the base element styles but before component-specific styles (around the section/container rules):

```css
/* =====================
   DIAGONAL SECTIONS
   Visual differentiator - suggests progress/momentum
   ===================== */

/* Diagonal bottom edge (content flows down into next section) */
.diagonal-bottom {
    position: relative;
    z-index: 1;
}

.diagonal-bottom::after {
    content: '';
    position: absolute;
    left: 0;
    right: 0;
    bottom: 0;
    height: 100px;
    background: inherit;
    transform: skewY(calc(-1 * var(--angle-degrees)));
    transform-origin: bottom left;
    z-index: -1;
}

/* Diagonal top edge (content flows down from previous section) */
.diagonal-top {
    position: relative;
    padding-top: calc(100vw * var(--angle-magic) + var(--space-10));
    margin-top: calc(-100vw * var(--angle-magic));
    z-index: 0;
}

.diagonal-top::before {
    content: '';
    position: absolute;
    left: 0;
    right: 0;
    top: 0;
    height: 100px;
    background: inherit;
    transform: skewY(calc(-1 * var(--angle-degrees)));
    transform-origin: top left;
    z-index: -1;
}

/* Combined: diagonal on both edges */
.diagonal-section {
    position: relative;
    padding-top: calc(100vw * var(--angle-magic) + var(--space-10));
    padding-bottom: calc(100vw * var(--angle-magic) + var(--space-10));
    z-index: 1;
}

.diagonal-section::before,
.diagonal-section::after {
    content: '';
    position: absolute;
    left: 0;
    right: 0;
    height: 100px;
    background: inherit;
    z-index: -1;
}

.diagonal-section::before {
    top: 0;
    transform: skewY(calc(-1 * var(--angle-degrees)));
    transform-origin: top left;
}

.diagonal-section::after {
    bottom: 0;
    transform: skewY(calc(-1 * var(--angle-degrees)));
    transform-origin: bottom left;
}

/* Purple diagonal variant */
.diagonal-purple {
    background: var(--color-primary);
    color: var(--color-text-inverted);
}

/* Light purple diagonal variant */
.diagonal-purple-light {
    background: var(--purple-50);
}
```
