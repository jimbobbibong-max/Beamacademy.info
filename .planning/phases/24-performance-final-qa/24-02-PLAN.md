- Or test on real device if available

**For each browser, verify:**
1. Page loads without errors
2. Hero displays correctly (tilted headline, texture)
3. Scroll down - sections animate in (or appear immediately if older browser)
4. Hover over CTAs - lift/shadow effects work
5. Click primary CTA - form/modal opens
6. Check console for JavaScript errors

**Known acceptable differences:**
- Older browsers (Safari <17.5, Firefox <129): Page load animation may not play, content appears immediately
- This is graceful degradation, NOT a bug

Report results for each browser tested. Mark as:
- PASS: Works correctly
- PASS (degraded): Works but without animations (acceptable)
- FAIL: Broken functionality or layout
