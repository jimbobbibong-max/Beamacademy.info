  <files>C:/Users/bskim/Downloads/beamacademy-index.html</files>
  <action>
Find the hero section CSS (look for .hero or hero-related classes).

Update the hero headline style to include the differentiator:

```css
/* Hero headline - THE DIFFERENTIATOR */
.hero h1,
.hero-title {
    font-family: var(--font-display);
    font-size: var(--text-hero);
    font-weight: 700;
    font-variation-settings: 'WONK' 1, 'SOFT' 50;
    line-height: var(--leading-tight);
    letter-spacing: var(--tracking-tight);
    color: var(--gray-900);
    transform: rotate(-2deg);
    transform-origin: left center;
    margin-bottom: var(--space-6);
}

/* Remove tilt on mobile - prevents overflow/cropping */
@media (max-width: 768px) {
    .hero h1,
    .hero-title {
        transform: none;
        font-variation-settings: 'WONK' 0.5; /* Reduced wonk for readability */
    }
}
```

Note: The font-variation-settings enables Fraunces' "wonky" character:
- WONK 1 = maximum quirky personality (desktop)
- WONK 0.5 = moderate quirk (mobile, for better readability)
- SOFT 50 = medium softness for warmth

If the hero uses a different class name (check the HTML), adjust the selectors accordingly.
