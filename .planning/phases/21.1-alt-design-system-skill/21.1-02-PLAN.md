
Note: We subset the weights to reduce file size:
- Fraunces: weights 500-700, WONK axis 0-1
- Bricolage: weights 400-600

ALSO add fallback font metrics to reduce CLS (add BEFORE the existing style block):
```css
/* Font fallback metrics to minimize CLS */
@font-face {
  font-family: 'Fraunces-fallback';
  src: local('Georgia');
  ascent-override: 95%;
  descent-override: 25%;
  line-gap-override: 0%;
  size-adjust: 105%;
}

@font-face {
  font-family: 'Bricolage-fallback';
  src: local('system-ui');
  ascent-override: 90%;
  descent-override: 22%;
  size-adjust: 102%;
}
```
