<task type="auto">
  <name>Task 1: Apply brutalist offset shadow to primary CTAs</name>
  <files>C:/Users/bskim/Downloads/beamacademy-index.html</files>
  <action>
Update primary CTA buttons with brutalist styling: hard offset shadow, no blur.

Find primary CTA button class (likely .btn-primary, .cta-button, or similar) and update:

```css
/* Brutalist CTA button styling */
.btn-primary,
.cta-button {
  /* Keep existing background/color or change to white for drama */
  background: var(--white, #fff);
  color: var(--purple-700, #7c3aed);
  border: 3px solid var(--gray-900, #111827);
  border-radius: var(--radius-md, 8px);
  
  /* Brutalist offset shadow - NO blur */
  box-shadow: 4px 4px 0 var(--gray-900, #111827);
  
  /* Smooth transitions for interactions */
  transition: transform 150ms ease, box-shadow 150ms ease;
}

/* Hover: lift up and increase shadow */
.btn-primary:hover,
.cta-button:hover {
  transform: translate(-2px, -2px);
  box-shadow: 6px 6px 0 var(--gray-900, #111827);
}

/* Active/pressed: push down */
.btn-primary:active,
.cta-button:active {
  transform: translate(2px, 2px);
  box-shadow: 2px 2px 0 var(--gray-900, #111827);
}

/* Focus visible for accessibility */
.btn-primary:focus-visible,
.cta-button:focus-visible {
  outline: none;
  box-shadow: 6px 6px 0 var(--gray-900, #111827), 0 0 0 3px var(--purple-300, #c4b5fd);
}
```

ALTERNATIVE: If white button feels too dramatic, keep purple background but add brutalist shadow:
```css
.btn-primary {
  background: var(--purple-600, #9333ea);
  color: white;
  border: 3px solid var(--gray-900, #111827);
  box-shadow: 4px 4px 0 var(--gray-900, #111827);
}
```

WHY 4px shadow: Industry standard for brutalist aesthetic. Not too subtle, not overwhelming.
WHY 150ms timing: Fast enough to feel responsive, slow enough to be perceivable (MOT-04).

2. Hover state lifts button and increases shadow
3. Click/active state pushes button down
4. Focus state shows both shadow and focus ring
